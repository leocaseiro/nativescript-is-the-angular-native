### Commands NativeScript CLI

```
tns create MyApp --ng
```

```
tns run android
```

```
tns run ios --emulator
tns livesync --emulator --watch
```

```
tns debug ios --chrome
```

```
tns prepare android
tns build android --uglify --snapshot
```

```
tns deploy android
```
